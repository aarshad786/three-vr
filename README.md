three-vr
===========

Device controls and render steps for mobile virtual reality applications

## Installation

```sh
$ npm install three-vr
```

## Usage

```js
// load three-vr extension
var threevr = require('three-vr');

// init rendering
threevr.init({
  renderer: renderer,
  camera: camera,
  scene: scene
});
```
